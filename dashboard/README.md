# Serene Scents Interactive Dashboard

Open `index.html` in Safari, Chrome, or Edge on macOS. The dashboard is styled in dark presentation mode with high-contrast KPI cards, glowing chart accents, interactive hover states, and fullscreen controls for projection.

No Power BI, internet connection, package installation, or local server is required because the dashboard is a self-contained HTML file with embedded data and charts.

## Meeting Flow

1. Start on **Overview** for the executive KPI story.
2. Move to **Trends** to explain daily revenue movement and seasonality.
3. Highlight the **January sales fall** KPI and monthly chart annotation to explain post-festive/New Year demand normalization.
4. Use **Forecast** to explain the basic model, advanced scenario, and calibration insight.
5. Use **Operations** for margin improvement, weekend demand, and delay risk.
6. Finish with **Recommendations** for founder-ready decisions.

## Projection Tips

- Click **Meeting mode** to enlarge dashboard typography.
- Click **Fullscreen** before projecting.
- Hover over chart elements for exact numbers during Q&A.
