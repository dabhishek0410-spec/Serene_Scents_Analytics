# Raw Data

This folder contains the project input CSV files copied from the original Serene Scents analysis workspace:

- `serene_scents_actual_growth.csv`: Daily actual operating performance.
- `serene_scents_advanced_forecast.csv`: Forecast generated from the advanced model with seasonality and festival boost logic.

The original Python forecasting and business analysis scripts are preserved in `src/`, and the original exploratory notebook is preserved in `notebooks/`.
