# Processed Data

Processed tables are generated by:

```bash
python src/serene_scents_pipeline.py
```

Key files:

- `monthly_kpis.csv`: Monthly revenue, profit, margin, growth, and unit metrics.
- `model_comparison.csv`: Forecast accuracy and bias metrics.
- `daily_forecast_comparison.csv`: Day-level actual-vs-forecast comparison.
- `festival_analysis.csv`: Festival vs non-festival performance.
- `cost_phase_analysis.csv`: Pre- and post-cost optimization economics.
